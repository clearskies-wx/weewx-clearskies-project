WMS-T Radar Animation PoC

Single L.TileLayer.WMS wrapped with L.timeDimension.layer.wms. The TIME parameter swaps per frame advance — no pre-rendered layers, no simultaneous server requests. Open DevTools Network tab to confirm: one set of tile requests per frame, not hundreds at once.

All 24 frames are fully pre-loaded before animation starts — the progress bar shows overall loading state, and playback begins only when every frame is cached. Looping is seamless because the cache holds all frames (nothing is evicted between loops).

Endpoint: mesonet.agron.iastate.edu/cgi-bin/wms/nexrad/n0r-t.cgi • Layer: nexrad-n0r-wmst • Last 2 hours, 5-min intervals (24 frames)
