1 · Hero weather icons — pick one (app-grade, none cartoony)

Same 8 conditions across each set. H1/H2 = Meteocons (multicolor, the set you liked) in two styles. H3/H4 = bold solid icons tinted per condition (sun=amber, rain=blue, storm=purple) — colorful and app-grade without being emoji; these can also be set to follow your theme/brand color instead. H1/H2 use the official @meteocons/svg-static npm package (v0.1.0, MIT) via jsDelivr — these are pure static SVGs with no SMIL animations. The Iconify meteocons: prefix serves the animated set whose rain/snow drops start at opacity:0 and animate in; when SMIL is broken or blocked they stay invisible, leaving only a bare cloud. The static package has no animation elements at all, so precipitation renders unconditionally. All four options here are static and reliable.
